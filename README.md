Swagger UI и Админ-панель / shutenkoadmin

Запуск
python app.py

Команды для PowerShell в Windows, чтобы остановить, удалить, собрать и запустить контейнер на основе `docker-compose.yml`

docker-compose stop
docker-compose rm -f
docker-compose build
docker-compose up -d

Описание:
- `docker-compose stop` — останавливает запущенные контейнеры, не удаляя их.
- `docker-compose rm -f` — удаляет остановленные контейнеры (флаг `-f` подавляет запрос на подтверждение).
- `docker-compose build` — пересобирает образ для сервиса, указанного в `docker-compose.yml`.
- `docker-compose up -d` — запускает контейнер в фоновом режиме.

Приложение доступно по адресу: http://localhost:5001/

Узнать структуру приложения:
    cd C:\Users\shute\development\shutenkoadmin
    tree /F
